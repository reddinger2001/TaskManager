USER MANAGEMENT

Accessing User Management

Navigate to Settings → Users from the header. Only admin users can access this page.

Creating a New User

On the Users page, fill in the Username and Password fields. Optionally check Admin to give the new user full access. Click Create User.

There are no email invites — the new user logs in directly with their credentials.

Resetting a Password

Admins can reset any user's password from the Users page. Enter a new password in the reset field next to the user and click Reset Password.

Regular users can change their own password by scrolling to the bottom of the Users page and using the Change My Password section.

Toggling Admin Status

Click the Toggle Admin button next to a user to switch them between admin and regular role. You cannot change your own admin status — there must always be at least one admin who can manage the system.

Deleting a User

When you delete a user, all their tasks, projects, and logs are reassigned to you (the deleting admin). Nothing is lost. You cannot delete your own account.

Activity Log

All user management actions (create, delete, role change, password reset) are logged in the System project. Check the project's logs for an audit trail.

TIP: If you're the only user, TaskManager works exactly like solo mode — just with a login screen. No configuration needed.
