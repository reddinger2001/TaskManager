PROJECTS

Projects group related tasks together. Tasks without a project go to the Inbox.

Creating a Project

Go to Projects and click the + button. Give it a name and optional description.

Project Health

On the dashboard, each project gets a card showing:

  • Progress bar — percentage of tasks completed
  • Next due date — the earliest upcoming deadline in that project
  • Task count — done vs. total

Nested Projects

Projects can have parent-child relationships. For example, "Q3 Planning" could be the parent of "Sprint 1", "Sprint 2", etc.

Project Board

Click a project to see its tasks in a Kanban-style board view, grouped by status.
