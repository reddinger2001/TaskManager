BACKUP & RESTORE

Your data lives in a single SQLite file. Backing up is as simple as copying that file.

Export

Go to Settings and click Download Database. This downloads the entire database as a taskmanager.db file.

Import

On the Settings page, use the Import Database section to upload a .db file. This replaces your current database.

Safety net: Before importing, the system automatically creates a backup of your current database with a timestamp. If the import fails, your data is restored from this safety backup.

Manual Backup

The database file lives at:

instance/taskmanager.db

You can copy this file directly. On macOS/Linux:

cp instance/taskmanager.db backups/backup-$(date +%Y%m%d).db

Migrating to Another Machine

To move your data to another computer:

  1. Export the database from Settings
  2. Copy the file to the new machine
  3. Place it at instance/taskmanager.db
  4. Start the app — your data is there

TIP: The database file is typically under 1MB for personal use. Even with thousands of tasks, it stays small.
