DEPENDENCIES

Task dependencies let you link tasks together so you can see what's blocking what.

How It Works

In the task detail view, there's a "Blocked By" dropdown. Select another task to create a dependency:

  • The dependent task shows a BLOCKED BY banner at the top
  • The blocking task shows a BLOCKS (N) section listing what depends on it
  • When the blocker is marked Done, the banner shows (resolved)

Circular Dependency Protection

The system prevents circular references. If Task A depends on Task B, you cannot set Task B to depend on Task A — the change will be rejected.

On the Dashboard

Tasks that block other tasks show an ⏳ N badge on the dashboard, where N is the number of tasks waiting on it.

Deleting a Blocking Task

If you delete the task that blocks something, the dependency is automatically removed (set to null). The dependent task is not deleted.
