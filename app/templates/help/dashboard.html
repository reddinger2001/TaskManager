DASHBOARD

The dashboard is designed for a 5-second pulse check. Read it top to bottom:

1. Big Numbers (Top Row)

Five cards showing your task counts at a glance:

  • Open — total tasks not yet done
  • Active — tasks you're currently working on
  • On Fire — P0 priority tasks and blocked tasks
  • Blocked — tasks with status "blocked"
  • Done — completed tasks

Each card is clickable — it takes you to the Tasks page filtered by that category.

2. Priority Strip

Thin horizontal bars showing the distribution of P0 through P3 tasks. If P0 or P1 is growing, you need attention there.

3. Project Health Cards

One card per project showing completion percentage and the next upcoming due date. Click a card to open that project.

4. Attention Panel (Left)

Three sections, in order of urgency:

  • Overdue / At Risk — tasks past their due date
  • Currently Active — what you're working on right now, with checkboxes to complete
  • Due This Week — upcoming deadlines

5. Status Panel (Right)

Three smaller sections:

  • Inbox — untriaged tasks with no project
  • Delegated — tasks assigned to others
  • Recently Done — recently completed tasks

GREEN MEANS GOOD: When a count is 0 (no fires, no blocks, empty inbox), it turns green with a checkmark. The goal is all green.
